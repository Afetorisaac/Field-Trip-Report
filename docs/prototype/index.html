About This Prototype

Procurement System Prototype

Author: Emmanuella Nana Ama Weir
Index: 1704895585
Institution: Korle-Bu Teaching Hospital

Prototype Features
  • Multi-role workflow simulation (Requester, Approver, Procurement, Finance)
  • Request creation and tracking
  • Approval/rejection workflow
  • Purchase order generation
  • Status tracking and notifications
  • Local storage persistence (data saved in browser)
Technology Stack
  • HTML5, CSS3, JavaScript (Vanilla JS)
  • Bootstrap 5 for UI components
  • LocalStorage API for data persistence
  • Responsive design for mobile and desktop
Note: This is a client-side prototype for demonstration purposes. A production system would include server-side processing, database storage, authentication, and security features.
